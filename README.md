# Simple-Power-BI-breakdown

Prosty projekt mający na celu wizualizację danych w Power BI. Wykorzystane dane to wyniki ankiety przeprowadzonej wśród specjalistów i osób związanych z branżą Data.

Surowe dane wymagały oczyszczenia i przekształcenia za pomocą Power Query. Proces ten obejmował m.in. standaryzację formatów oraz zmianę typów danych, aby umożliwić ich czytelną prezentację na wykresach.
